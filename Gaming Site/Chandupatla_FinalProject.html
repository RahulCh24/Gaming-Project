About My Website

I am interested in this project because I spent a lot of my childhood playing video games. I consistently finished first among my buddies in all games. I've participated in numerous online competitions and used to play professionally in PUBG. I therefore choose this subject for my project. Top gamers, popular games, trending games, categories, links to gaming websites, and login pages are among the topics covered. This website was built using HTML and CSS.

My Experience

Creating this website has taught me several fundamental concepts in html and CSS. I want to increase my toolbox with things like javascript, sql, and web design frameworks. Since this was my first website, I'd like to improve upon it and make it more attractive. Some CSS attributes, including float and the distinction between margin and padding, proved to be difficult for me to understand during this session. I believe adding a small amount of javascript to the course syllabus may definitely make web development more engaging.
